## Descripcion del trabajo

El trabajo trata de un analisis de datos provenientes del gobierno de Mexico, donde se evaluan distintos indicadores para determinar la calidad del agua subterranea. A traves de distintos indicadores de sustancias quimicas y biologicas determina la calidad clasificando como un semaforo (verde, amarillo, rojo)

Tambien se crearon modelos de clasificacion KNN y Random Forest para clasificar en los colores del semaforo segun los valores de los indicadores

https://www.gob.mx/conagua/es/articulos/indicadores-de-calidad-del-agua?idiom=es

Busque estos datos ya que para la tesis de Maestria trato un tema similar, la aplicacion de tecnologias (Blockchain, IoT) para el manejo de informacion del agua. Por lo que buscaba que el trabajo me ayude a ampliar los conocimientos en este tema
